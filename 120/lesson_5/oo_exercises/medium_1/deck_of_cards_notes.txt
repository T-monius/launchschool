# deck_of_cards_notes.txt

Understanding
  > A deck has 52 cards
  > The Card class has rank and value instance variables
    which are accessible through a getter method outside of the class.
  > Deck class knows all of the possible ranks and suits
    - Constants
  > Instantiate a deck by iterating through the raks and
    suits and creating a card for each on.
  > The deck ought to be reset when a new deck object is
    instantiated, and when all of the cards in a deck are
    drawn.
    - reset instance method
      < called in constructor
      < called if draw instance method empties the deck
  > Draw instance method
    - removes a single card from the deck at random
    - calls reset if the deck is empty
