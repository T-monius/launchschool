# reverse_array2_notes.txt

Understanding
  - Input
    > Array
  - Task
    > Reverse the array
    > Do not mutate
  - Output
    > New array
    > Same size as the input array
    > Elements in reverse

Implementation
  - Instantiate an empty return array
  - Iterate the input array in reverse order
    > Push the element at iteration to the return array
    > Return the new array
