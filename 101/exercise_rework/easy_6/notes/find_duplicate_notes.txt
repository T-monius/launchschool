Understanding
  - Input
    > Arr
      + Unordered
      + All elements are integers
      + All but one occur exactly once
  - Task
    > Find the element that occurs twice
  - Output
    > Integer

Implementation
  - Order the array
  - Iterate the array looking at the current element and the
    next
  - If the current element and the following match, then return
    the current element.
