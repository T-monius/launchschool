# swap_case_notes.txt

Understanding
  - Input
    > Single argument
    > String
  - Task
    > Swap the case of all alpha-numeric characters
  - Output
    > A new string
    > Letters swapped in case

Implementation
  - Iterate over the string
  - Transformation
  - Hash
    > Lowercase letters map to uppercase
    > Uppercase to lowercase
  - Ternary
    > If lowecase return uppercase version
    > Else return lowercase version
    > Works with special characters since they will be
      unaffected by a call to 'String#downcase'
