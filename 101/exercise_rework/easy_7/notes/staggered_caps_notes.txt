# staggered_caps_notes.txt

Understanding
  - Input
    > Single argument
    > String
  - Task
    > Transform the input string
    > Capitalize every other character
    > Lowercase every 2nd
  - Output
    > New String
    > Every other character of the input sting is capitalized
    > Even indexes
    > Odd indexes lowercase

Implementation
  - INstantiate a return string
  - Iterate the input string
    > If the index is even, capitalize the character
    > Else downcase it
    > Push the character to the return string
  - Return new string
