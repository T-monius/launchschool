# two_lists_notes.txt

Understanding
  - Input
    + Two arguments
    + Arrays
    + Same number of elements
  - Task
    + Combine the two arrays
    + Interleave the elements
  - Output
    + An array
    + Elements from the input arrays interleaved

Implementatin
  - Iteration
  - Instantiate a return array
  - Iterate one of the arrays
    > Access the element at iteration
    > Access the corresponding index
    > Push the element from iterated array (first argument) to
      the return array
    > Push the corresponding indexed element from the second
      second array (second argument) to the return array
    > Continue iterating
  - Return array
